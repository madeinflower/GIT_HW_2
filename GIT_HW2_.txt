
1. git checkout -- <имя_файла>
2. git clean  —xdn, а затем git clean  —xdf
3. git checkout -- <путь_к_файлу>
4. - git revert <идентификатор_коммита>
   - git reset
5. git commit --amend -m "New good comment"
6. - внесите необходимые изменения в файлы вашего проекта
   - git add <файлы> для добавления измененных файлов в индекс
   - git commit --amend для исправления последнего коммита
   - сохраните изменения в сообщении коммита и закройте редактор
7. Изменение истории коммитов, которые уже выгружены на сервер Git, не рекомендуется, поскольку это может вызвать проблемы синхронизации и привести к разделению истории. Вместо этого рекомендуется использовать другие подходы, такие как git revert, чтобы отменить изменения в отдельном коммите, или согласование с другими разработчиками для корректировки истории, если это действительно необходимо.
